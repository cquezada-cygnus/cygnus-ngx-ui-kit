Bienvenido al Cygnus UI Kit

El Cygnus UI Kit es un conjunto de componentes diseñados para crear aplicaciones web consistentes y profesionales. Este kit incluye botones, inputs, tipografía, colores y más, todo basado en Tailwind CSS.

¿Qué encontrarás aquí?

  • Componentes reutilizables para construir interfaces de usuario rápidas y consistentes.
  • Estilos personalizables para adaptarse a la identidad de tu marca.
  • Documentación clara para que puedas integrar cada componente fácilmente.

Cómo comenzar

Para comenzar, explora los componentes desde el menú izquierdo y selecciona lo que necesitas. Cada componente tiene ejemplos y configuraciones que puedes personalizar.
