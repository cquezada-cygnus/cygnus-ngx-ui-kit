Dropdown Menu

El menú desplegable muestra una lista de opciones en superficies temporales. Aparece cuando los usuarios interactúan con un botón, una acción u otro control. Cuando un usuario hace clic o pasa el cursor sobre el elemento principal, aparece el menú desplegable, que presenta una lista de acciones apiladas verticalmente.
