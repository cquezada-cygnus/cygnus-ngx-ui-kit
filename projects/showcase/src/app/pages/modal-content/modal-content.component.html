Modales

Se unan las ventanas modales para informar a los usuarios sobre tareas o información importante que requieren decisiones o implican múltiples pasos. Un modal es una ventana que muestra información crítica, deshabilita todas las funciones de la aplicación cuando aparece y permanece en la pantalla hasta que se Confirmara o se descarta. Vea a continuación nuestros ejemplos modales diseñados con Tailwind CSS que puede usar para sus aplicaciones web o móviles.

Cómo usar cygnus-modal

Importar el componente de la biblioteca ngx-cygnus-ui y algún otro componente que podrías necesitar para llamar al modal en el archivo .ts del componente donde lo vas a implementar.

Implementar una o dos variables tipo boolean y sus respectivos métodos para manejar el modal y el botón X de cierre si corresponde.

En el documento .html del componente, posicionar la plantilla del modal, el componente que se requiera para ejecutar el modal y agregar la configuración según corresponda.
  • [modalTitle]: Acá se indica el texto correspondiente al título del modal.
  • [modalContent]: Acá se indica el texto correspondiente al contenido del modal.
  • [(showModal)]:Acá se entrega una variable tipo boolean para manejar la aparición/desaparición del modal.
