Badges

Badges (también conocido como Insignias), es un elemento compacto que representa una entrada, un atributo o una acción. Este elemento aparece de forma dinámica como un grupo de múltiples elementos interactivos y permite a los usuarios introducir información, realizar selecciones, filtrar contenido o activar acciones. A menudo vienen con la capacidad de ser removidos o interactuar con ellos, lo que permite a los usuarios manipular los datos o la selección representados por el badge.

Cómo usar badge simple

Importar el componente de la biblioteca ngx-cygnus-ui en el archivo .ts del componente donde lo vas a implementar. Es posible que necesites importar también CygnusButtonComponent dependiendo del tipo de badge.

En el documento .html del componente, posicionar la plantilla de la tabla y agregar la configuración según corresponda.

[badgeTypes]: acá se indica el tipo de badge que se quiere utilizar dependiendo del tamaño, color, etc. También pueden combinarse varios aunque se pueden sobreescribir algunas propiedades.

  • 'badge-xs': Badge del tamaño más pequeño.
  • 'badge-sm': Badge pequeño.
  • 'badge': Badge tamaño estándar.
  • 'badge-lg': Badge grande.
  • 'badge-primary': Badge color estándar.
  • 'badge-gray': Badge gris.
  • 'badge-error': Badge color rojo.
  • 'badge-warning': Badge color amarillo.
  • 'badge-success': Badge color verde.
  • 'badge-pink': badge color rosado.
  • 'badge-purple': Badge color morado.
  • 'badge-indigo': Badge color indigo.
  • 'badge-icon': badge que lleva un ícono.
  • 'badge-icon-only': Badge que sólo lleva un ícono, sin texto.
  • 'badge-btn': Badge combinado con un botón x el cual elimina el badge al hacer click en él. Tiene apariencia de desactivado. Efecto hover.
  • 'badge-btn-outlined': Badge combinado con un botón x el cual elimina el badge al hacer click en él. Tiene apariencia de botón azul. Efecto hover.
  • 'badge-state-success': Badge para expresar estado éxito u online.
  • 'badge-state-error': Badge para expresar estado error u offline.

[badgeText]: acá se indica el texto que llevará el badge.
