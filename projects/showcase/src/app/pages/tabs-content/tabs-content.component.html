Pestañas

Las pestañas responsivas crean una jerarquía de navegación secundaria para su sitio web. Las pestañas son componentes que representan y muestran subsecciones a los usuarios. Organizan el contenido en categorías para facilitar el acceso y lograr una aplicación más ordenada. Las pestañas también permiten a los usuarios cambiar entre diferentes vistas o secciones de contenido dentro de la misma ventana o espacio. Pueden ser horizontales o verticales, según los requisitos de diseño. El diseño visual de las pestañas suele incluir una indicación clara de qué pestaña está activa en ese momento, a menudo mediante color, resaltado o subrayado.

Cómo usar cygnus-tabs

Importar el componente de la biblioteca ngx-cygnus-ui y la interfaz Tab en el archivo .ts del componente donde lo vas a implementar.

Añadir un arreglo de elementos que usen la interfaz Tab para mostrar.

En el documento .html del componente, posicionar la plantilla y agregar la configuración según corresponda.
  • [tabsArr]: Aquí se debe indicar el arreglo de tabs que se utilizará.
  • [showIcon]: Acá se indica con un boolean si se mostrará un ícono en los tab o no.
  • [showTabInnerHTML]: Acá se indica con un boolean si se ingresará por cada tab Html con contenido adicional.
