Dropdown Menu

El menú desplegable muestra una lista de opciones en superficies temporales. Cuando un usuario hace click sobre el elemento principal aparece el menú desplegable que presenta una lista de acciones apiladas verticalmente.

Cómo usar Dropdown Menu

Importar el componente de la biblioteca ngx-cygnus-ui y la interfaz DropdownItemData en el archivo .ts del componente donde lo vas a implementar. Debes crear un arreglo con todos los elementos que se quieran desplegar según la estructura de la interfaz.
  • iconAssetText: Debes entregar un string con la url o nombre del ícono dentro de la biblioteca ngx-cygnus-icons que indique la dirección o nombre del ícono que quieres mostrar para la opción dentro del dropdown.
  • routerLinkText: Url a la cuál se redireccionará cuando elijas una opción del dropdown. Si la dejas con el valor './' permanecerá en la misma página.
  • itemText: Texto que se mostrará en la opción del dropdown.

En el documento .html del componente, posicionar la plantilla del menú y agregar la configuración según corresponda.
  • dropdownMenuTitle: Debes entregar un string con el título del menú dropdown.
  • dropdownItemType: Debes entregar un string que indique el tipo de menú dropdown. Pueden ser: "simple", "radiobutton", "iconText", "close", "breadcrumb".
  • dropdownItemDataArr: Acá debes entregar un arreglo de elementos DropdownItemData para que se desplieguen en el menú.
