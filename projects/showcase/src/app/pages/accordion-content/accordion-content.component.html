Acordeón

El acordeón es un componente de interfaz de usuario que permite al usuario mostrar y ocultar secciones de contenido relacionado en una página. Hay muchas formas de utilizar este componente, como mostrar una lista de preguntas frecuentes, mostrar varios menús y submenús, mostrar las ubicaciones de una empresa en particular, etc.

Acordeón simple

Se puede utilizar íconos personalizados para el elemento disparador del acordeón y, mediante el selector group-open de atributos en Tailwind CSS, puede controlar el ícono para los estados abierto y cerrado del acordeón.
