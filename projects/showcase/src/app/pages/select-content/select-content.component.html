Cómo usar cygnus-select

Importar el componente de la biblioteca ngx-cygnus-ui y la interfaz SelectGeneric en el archivo .ts del componente donde lo vas a implementar.

Crear un arreglo de elementos que cumplan con la interfaz SelectGeneric en el documento .ts.

En el documento .html del componente, posicionar la plantilla del selector y agregar la configuración según corresponda.
  • [control]: Acá puedes ingresar un control de formulario en caso de que quieras integrar el valor del select en un formulario: <cygnus-select [control]="form.controls.selExample" />.
  • [options]: Acá debes entregar el arreglo con items para mostrar las opciones del selector.
  • [selSize]: Acá puedes indicar el tamaño del selector. Corresponde a un string que puede ser 'lg' o 'xs'. Si no se indica nada, tiene un tamaño estándar.
  • [isDisabled]: Indica si el selector está o no deshabilitado.
  • [selState]: Si se utiliza es para dar un estilo rojo en caso de error o un estilo verde en caso de éxito. Acepta un string con los valores 'error' o 'success'.
  • [selectLabel]: Acá se indica un texto para usar sobre el selector.
  • [selectHint]: Acá se indica un texto para usar debajo del selector.
