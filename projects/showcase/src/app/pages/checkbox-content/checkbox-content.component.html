Checkboxes

Cómo usar los checkboxes

Importar el componente de la biblioteca ngx-cygnus-ui en el archivo .ts del componente donde lo vas a implementar.

En el documento .html del componente, posicionar la plantilla de la tabla y agregar la configuración según corresponda.

Se puede añadir configuración adicional para incluir en formulario.

Checkboxes tamaños

Se indica en [checkboxSize] un string que indica el tamaño. Dos tamaños disponibles: 'sm' y 'lg'.

Si no se indica el tamaño, por defecto es 'sm'

Checkbox con texto

Se debe indicar en [checkboxType] el string 'text' para que el checkbox renderice un. texto. Cuando no se indica [checkboxType] por defecto tiene el valor 'base' que significa que se encontrará el checkbox solo sin texto. En [checkboxText] se entrega un string con el texto que se requiera.

Checkbox con texto y texto de ayuda

Se debe indicar en [checkboxType] el string 'text-hint' para que el checkbox renderice un texto de indicación y un texto de ayuda. Cuando no se indica [checkboxType] por defecto tiene el valor 'base' que significa que se encontrará el checkbox solo sin texto. En [checkboxText] se entrega un string con el texto que se requiera como indicación y en [checkboxHint] se indica el texto de ayuda.
