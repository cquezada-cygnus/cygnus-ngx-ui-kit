Colapsable predeterminado

Usa el componente colapsable CSS Tailwind para permitir al usuario mostrar y ocultar secciones de contenido relacionado en una página. Hay muchas formas de utilizar este componente, como mostrar una lista de preguntas frecuentes, mostrar varios menús y submenús, mostrar las ubicaciones de una empresa en particular, etc.

Utilice este colapsable hecho en Tailwind. Puedes utilizarlo para acordeón, elementos plegables y mucho más.
